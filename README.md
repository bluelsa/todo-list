Todo list
====
此專案使用React框架製作Todo list頁面


安裝
--
1. 開啟Terminal終端機，將專案clone到本機伺服器<br>
```
git clone https://github.com/bluelsa/alpha-shop
```
2. 初始<br>
* 切至專案資料夾
```
cd 
```
* 安裝套件<br>
```
npm install
```
3. 開啟程式<br>
```
npm start 
```
```
npm run dev-server
```

terminal 顯示網址 http://localhost:3000 表示啟動成功，打開瀏覽器進入此網址

環境建置需求
--
* Node.js 14.18.1
* React 18.2.0
